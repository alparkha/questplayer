# 나자신 키우기: The Quest Player

일상을 RPG처럼 즐겁게 만드는 게임형 스케줄러!

## 주요 기능

- 🎮 RPG 스타일의 일정 관리
- ⚔️ 일정 완료시 화려한 애니메이션
- 📈 레벨업 시스템
- 🏆 업적 시스템
- 💾 진행 상황 저장

## 시작하기

1. 저장소를 클론합니다:
```bash
git clone https://github.com/[사용자명]/quest-player.git
```

2. 웹 브라우저에서 `index.html`을 엽니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (ES6+)
